# 🌐 Basic Static Website

A responsive multi-page website built using HTML and CSS.

## 🚀 Features

- Responsive Design
- Sticky Navigation Bar
- Hero Section
- About Page
- Contact Page
- Feature Cards
- Footer
- Clean and Modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Design (Media Queries)
